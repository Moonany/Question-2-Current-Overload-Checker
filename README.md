# Current Overload Checker

A simple C++ console program that checks whether a circuit's measured current exceeds its rated breaker current.

## Description

This program:
- prompts the user for a circuit breaker's rated current (in amperes)
- prompts for the measured current in the circuit (in amperes)
- compares the two values
- displays whether the circuit is overloaded or within safe limits

## Files

- `Currebt Overload Checker.cpp` - source code for the program
- `Makefile.win` - Windows makefile generated by Dev-C++
- `Current Overload Checker.exe` - compiled executable
- `current_overload_checker_flowchart.png` - flowchart of the program logic

## Build

### Option 1: Using `g++`

Open a terminal in the project folder and run:

```sh
"C:\Program Files (x86)\Dev-Cpp\MinGW64\bin\g++.exe" "Currebt Overload Checker.cpp" -o "Current Overload Checker.exe"
```

### Option 2: Using `Makefile.win`

If you have `make` installed, run:

```sh
make -f Makefile.win
```

## Run

From the project folder, execute:

```sh
"Current Overload Checker.exe"
```

Then enter the rated current and measured current when prompted.
